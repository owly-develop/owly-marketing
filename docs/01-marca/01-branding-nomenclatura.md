# 🎨 Branding y Nomenclatura

> Reglas oficiales de marca. Extraído de `owly-sales-ecosystem/docs/02-architecture/07-branding-guidelines.md`. **Aplica a TODO el marketing**: landing pages, redes, emails, ads, presentaciones.

## Identidad de marca

- **Nombre**: OWLY
- **Dominio**: owlysales.ai
- **Tagline**: Sales Platform
- **Descripción**: Plataforma integral de ventas con inteligencia artificial.
- **Logo / símbolo**: 🦉 (búho — inteligencia, visión nocturna = insights, sabiduría).

---

## ⛔ Regla #1 — NUNCA usar "CRM"

OWLY **no** se comunica como un CRM. Está prohibido usar "CRM" en UI, marketing, ads, landing pages, emails, onboarding o help center.

| ❌ NO usar | ✅ Usar en su lugar |
|-----------|--------------------|
| CRM | Sales Platform / Plataforma de ventas |
| Real Estate CRM | Sales Platform for Real Estate |
| CRM Tool | Sales Tool / Sales Platform |
| CRM System | Sales System / Plataforma OWLY |
| Panel CRM | Panel de ventas / Sales Dashboard |
| Vista CRM | Vista de gestión |

> Posicionamiento: OWLY es una **plataforma integral de ventas inmobiliarias**, no un CRM genérico.

---

## Nomenclatura por contexto

- **SEO / meta**: `OWLY - Sales Platform` · "Plataforma integral de ventas inmobiliarias con IA".
- **Títulos**: "Dashboard OWLY", "OWLY - Gestión de Leads", "OWLY - Comunicación Omnicanal".
- **Estructura de header**: `{Nombre del módulo} OWLY` + descripción específica (sin "CRM").

Ejemplos por módulo:
- Dashboard → "Panel de control de ventas"
- Leads → "Gestión de oportunidades"
- Comunicación → "Centro de comunicación omnicanal"
- Radar → "Inteligencia de ventas"
- Analytics → "Análisis y reportes"
